I, Object!

Constructors

  • A function called with the "new" keyword that returns an object
  • Capitalized by convention

Classes

  • We don't need no stinking classes
  • You can build a "classical" inheritance model if you really want
  • But you really don't

Prototypical Inheritance

  • Every object has a special "prototype" property
  • Accessing a method or property of an object checks the object first, then the prototype
  • Because the prototype itself is an object, the chain can be extended
  • All objects created with the same constructor share a prototype

More Prototypes

The jQuery Function

  • jQuery() or $()
  • Behaves differently based on type of input
    • DOM Elements => Wrap in jQuery set
    • String => Select elements OR create elements from HTML
    • Function => Register for document.ready
  • Return value is usually another jQuery set, which enables "method chaining"
